# Workout Buddy

### Start 
1. To start, run `npm install` on both the backend and frontend folders.
  ```
  ; cd backend
  ; npm install
  ; cd frontend
  ; npm install
  ```
## To run the application 
2. Start by running the backend server `npm run dev`
  ```
  ; cd backend
  ; npm run dev
  ```
3. Then cd into the frontend folder and run `npm start`
  ```
  ; cd frontend
  ; npm start
  ```
